1. 对象
    1) 什么是对象？
        对象是一组属性和方法组成的集合，对象是复合类型的数据

    2) 对象分类
        内置对象：也称为系统对象，系统自带的对象，只管使用不管定义
        自定义对象：程序员自己编写的对象

    3) 定义对象
        a. 实例化方式
            let obj = new Object();

        b. 字面量方式
            let obj={};
            let obj={属性名:属性值, 属性名2:属性值, ...};

    4) 对象的属性和方法操作
        a. 新增属性
            obj.新属性名=值;
            obj['新属性名']=值; //动态属性

        b. 修改属性
            obj.属性名=新的值;

        c. 删除属性
            delete obj.属性名;

        d. 查询属性
            obj.属性名;
            obj['属性名'];

    5) 遍历对象
        for(let key in obj){
            key     遍历到的属性名
            obj[key] 属性值
        }

    6) 方法中的this
        对象方法中的this，指向对象自身

2. String字符串对象
    1) 什么是字符串对象？
        广义：所有的数据都可以当成对象，可以调用属性和方法（一切皆对象）
        狭义：使用new创建出来的字符串对象

    2) 字符串的属性和方法
        .length     获取字符串的长度
        .indexOf()  查找字符串在另一个字符串中首次出现的位置，存在返回下标，不存在返回-1
        .replace()  替换字符串，返回替换后的字符串
        .substr()   截取字符串，返回截取到的子字符串
        .split()    拆分字符串，返回拆分后得到的数组

        .toLowerCase()  变小写
        .toUpperCase()  变大写

3. Number对象
    .toFixed()      保留指定位数的小数 

4. Math对象
    .PI         圆周率
    .ceil()     向上取整
    .floor()    向下取整
    .max()      最大值
    .min()      最小值
    .random()   随机小数，包含0不包含1

5. Date对象
    1) 创建日期对象
        //创建当前日期对象
        let d=new Date();

        //根据日期字符串创建对象
        let d=new Date('2022-05-01 00:00:00');

        //根据数字参数创建对象
        let d=new Date(2022, 4, 1, 0, 0, 0);

    2) 日期对象的方法
        .getTime()  获取时间戳，单位：毫秒
