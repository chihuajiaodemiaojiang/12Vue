1. 函数
    1) 什么是函数？
        函数就是对一段实现功能的代码进行的封装，可以重复使用

    2) 定义函数
        a. 函数声明
            function 函数名(形式参数1, 形式参数2, 形式参数n){
                函数体;
                return 值;
            }

        b. 函数表达式
            let 变量名=function(形式参数1, 形式参数2, 形式参数n){
                函数体;
                return 值;
            }

    3) 函数调用
        函数名(实际参数1, 实际参数2, 实际参数n);

    4) 函数参数
        a. 形式参数
            形式参数出现在函数定义的小括号内，用于接收传入的实际值

        b. 实际参数
            实际参数出现在函数调用的小括号内，用于向函数传递实际值

        c. 函数参数的默认值
            ES5 利用短路运算实现默认值
                a=a||0;

            ES6 使用新语法实现默认值
                function 函数名(a=0, b=0){
                    ...
                }

    5) 函数返回值
        函数可以设置返回值，将函数运行的结果返回到【调用处】
        return 值;

    6) 函数定义三部曲
        确定函数名
        确定参数个数
        确定返回值

2. 变量作用域
    1) 什么是作用域？
        作用域是指变量起作用的范围

    2) 作用域划分
        由于函数的封装特性，产生独立的运行环境，将作用域划分为：
            全局作用域：在函数外定义的变量是全局变量，可以在任意地方使用
            局部作用域：在函数内定义的变量是局部变量，只能在当前函数内使用

3. 系统函数
    Number()    转数字
    String()
    Boolean()
    parseInt()
    parseFloat()

    isNaN()     判断是不是数字，不是数字返回true，是数字返回false
